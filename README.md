### Downloading backups via FTP

Reads yaml file - info.yaml\
Sample file content:
~~~~~~~~
server1:
    address: <ip>
    username: <username>
    password: <password>
    source: <path to directory in ftp>
    destination: <path to local directory>
server2:
    address: <ip>
    username: <username>
    password: <password>
    source: <path to directory in ftp>
    destination: <path to local directory>
~~~~~~~~
